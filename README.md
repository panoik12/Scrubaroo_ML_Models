# Scrubaroo — ML Pipeline

This repository contains the machine learning experiments for **Scrubaroo**, a Web3 application that combines deep learning and Sui Blockchain to incentivize citizens for environmental cleanup actions.

- **Backend:** https://itsmepanoik-scrubaroo-api.hf.space
- **Paper:** Scrubaroo: AI-Powered Waste Detection and Blockchain-Based Citizen Reward System, SETN 2026
- **Live App:** https://scrubaroo.vercel.app/
  
## Overview

Since the dataset lacked waste density labels, three automatic labeling strategies were developed and compared:

| Strategy | Feature Extractor | Images | Best Model Accuracy |
|---|---|---|---|
| KMeans-ResNet50 | ResNet50 (2048-dim) | 3,059 | 89.54% (MNV2) |
| KMeans-MobileNetV2 | MobileNetV2 (1280-dim) | 3,059 | **91.34% (MNV2 + WL)** |
| CLIP K-Means Hybrid | CLIP ViT-B/32 (512-dim) | 2,255 | 89.62% (MNV2) |

## How to Run

### Step 1: Clustering (Label Generation)

Run one of the clustering notebooks to generate the labeled dataset:

```
clustering/KMeans_WithMobileNetV2.ipynb   ← Recommended
```

Update the `root` path to point to your image folder:
```python
all_dataset = torchvision.datasets.ImageFolder(
    root="YOUR_PATH/AllImages/All", ...)
```

### Step 2: Training

Run the desired training notebook. Example for best model:

```
training/WeightLoss_MobileNetV2_KMeansMobileNet.ipynb
```

Update dataset paths:
```python
train_dataset = torchvision.datasets.ImageFolder(
    root="YOUR_PATH/KMeans_Dataset_with_MobileNetV2/train", ...)
test_dataset = torchvision.datasets.ImageFolder(
    root="YOUR_PATH/KMeans_Dataset_with_MobileNetV2/test", ...)
```

### Step 3: Prediction

```python
def predict_confidence(image_path):
```
